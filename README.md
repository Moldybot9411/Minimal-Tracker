# Minimal Tray Tasker

Track your everyday tasks right from your system tray

<img src="./images/Screenshot.png" alt="demo" width="300"/>


## Install

Head to the releases page and download the right package for your system.

## Developing

Install Node dependencies
```
npm i
```

Run the project
```
npm run tauri dev
```

Build the project
```
npm run tauri build
```
